# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.7.0] - 2025-12-29

### Added

- Integration of eIQ Neutron SDK 2.2.2 and Neutron Conversion for 25.06/25.09 SDKs
- Conversion targets for i.MX943, i.MX952, MCX-N54x, MCX-N94x, S32K5, S32N79
- Examples for NXP-specific additional workflows
- Mapping of Neutron Conversion to Linux Yocto BSP releases
- TFLite model loading from directory

### Fixed

- Output model path for Vela Conversion
- Small fixes and refactoring
